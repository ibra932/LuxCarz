# LuxCarz 🚗

Le projet couvre l'ensemble des fonctionnalités d'une plateforme de location : consultation du parc automobile, gestion des réservations et administration des données — le tout relié à une base de données relationnelle.

## Fonctionnalités

- 🚘 **Catalogue de véhicules** — consultation et recherche des voitures disponibles à la location
- 📋 **Gestion des réservations** — création, modification et suppression de réservations
- 🛠️ **Interface d'administration** — gestion complète des données via les méthodes HTTP (GET, POST, PUT, DELETE)
- 🗄️ **Base de données relationnelle** — modélisation et gestion des données avec MySQL via phpMyAdmin

## Stack technique

| Technologie | Rôle |
|---|---|
| HTML / CSS | Structure et mise en forme des pages |
| JavaScript | Interactions côté client |
| PHP | Logique serveur et traitement des requêtes |
| MySQL / phpMyAdmin | Base de données relationnelle |


Objectif : concevoir et développer une application web full-stack fonctionnelle de A à Z.
